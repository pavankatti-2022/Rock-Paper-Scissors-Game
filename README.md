# Rock Paper Scissors Game

A simple Rock Paper Scissors game built using **HTML, CSS, and JavaScript**.  
The player competes against the computer, and scores are stored using browser local storage.

## Features

- Play Rock, Paper, Scissors against the computer
- Random computer choice generation
- Live score updates
- Score persistence with localStorage
- Reset score functionality
- Responsive and simple UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How It Works

1. User selects Rock, Paper, or Scissors.
2. Computer randomly selects one option.
3. Winner is decided based on game rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. Scores update automatically.
5. Scores are saved in localStorage, so they remain after page refresh.


## Game Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice results in a draw
